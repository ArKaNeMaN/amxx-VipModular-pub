# ItemsList

Список любых предметов.

## Пример

```jsonc
{
    "Type": "ItemsList",

    "Items": [
        {
            "Type": "Cwapi",

            "Name": "Vip_Ak47",
            "GiveType": "Replace"
        },
        {
            "Type": "Cwapi",

            "Name": "Vip_Deagle",
            "GiveType": "Replace"
        }
    ]
}
```

## Параметры

- `Items`
  - Массив обьектов предметов.
