# VipInTab

## Пример

```json
{
    "Module": "VipInTab",

    "Enabled": true,
    "Override": true
}
```

## Параметры

- `Enabled` - будет ли отображаться статус VIP в таблице счёта.
  - _Если честно, хз зачем этот параметр нужен :)_
- `Override` - будет ли статус VIP перекрывать остальные статусы типа `Bomb`, `Dead` и т.д.
  - `true` - будет;
  - `false` - не будет.
